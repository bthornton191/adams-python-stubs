    objects instance-attribute ¶
    objects: List[ObjectBase]

    All child objects created by this UDE instance (Parts, Markers, Forces, etc.).

    This list can be used as an alternative traversal path when model.Parts misses parts due to short-name collisions between UDE instances (ADMS-137078).

    Example::

    from Adams import Part
ude_parts = [o for ude in model.UserDefinedInstances.values()
             for o in ude.objects if isinstance(o, Part)]
